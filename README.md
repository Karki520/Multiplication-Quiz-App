# Multiplication Quiz App

A simple interactive quiz game built during my MCA to practice JavaScript fundamentals. It generates random multiplication questions and tracks the user's score.

## How to Play
1. Open `index.html` in any browser
2. Read the multiplication question displayed
3. Enter your answer in the input box and click Submit
4. Correct answers increase your score

## Technologies Used
- **HTML** - For page structure and form elements
- **CSS** - For basic styling in `app.css`
- **JavaScript** - For game logic in `app.js` including random number generation, DOM manipulation, and event handling

## Key Concepts I Practiced
1. DOM Manipulation using `document.getElementById()`
2. Preventing form reload with `event.preventDefault()`
3. Generating random numbers using `Math.random()`
4. Updating UI dynamically based on user input

## How to Run Locally
Just download the files and open `index.html` in your browser. No installation needed.
